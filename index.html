Google Doc to WordPress Formatter

How to use

On first run, your browser may ask to allow clipboard access. You will need to accept for this tool to work.

  1. Copy text from Google Docs.
  2. Press control + v on this page (or command + v for MacOs). The formatter will run automatically and replace the rich text in your clipboard with a cleaned up html version that WP can use.
  3. Paste clipboard into the WP editor in text mode. At first it will appear as an ugly blob of text, but if you swith to visual mode it will re-structure and subsequent text-mode views will now appear correct.

Problems this solves

  • Removes unnecessary tags and attributes
  • Removes strong tags applied to headers
  • Keeps bolds and italics intact
  • Automatically opens external links in new tabs
