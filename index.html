Google Doc to HTML Formatter(BETA VERSION)

How to use

On first run, your browser may ask to allow clipboard access. You will need to accept for this tool to work.

  1. Copy text from Google Docs.
  2. In the top left corner of this app, set the toggle to the correct website. Your setting will be saved on your browser until the cookies are cleared.
  3. Press control + v on this page (or command + v for MacOs). The formatter will run automatically and replace the rich text in your clipboard with a cleaned up HTML version.
  4. Paste into the CMS editor. For Wordpress make sure your pasting into text mode. In Storyblok paste into the source code view.

Problems this solves

  • Removes unnecessary tags, attributes, and bloat
  • Removes strong tags applied to headers
  • Keeps bolds and italics intact
  • Automatically opens external links in new tabs
