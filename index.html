Google Doc to WordPress Formatter

Instructions:

  1. Press control + v
  2. Press control + a
  3. Press control + c
  4. Paste into WP visual editor

What does this do?

  • removes unnecessary span tags
  • keeps rich text formatting (bolds, italics, etc.)
  • adds target="_blank" rel="noopener" to external links
  • removes aria attributes from list items
  • removes double carriage returns
  • partially pre-formats tables
